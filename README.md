# NFT Mint & Rarity Alerts Bot

Telegram bot delivering real-time alerts on early NFT mints, rarity highlights, whitelist opens, floor movements, and rare secondary sales for verified projects.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
